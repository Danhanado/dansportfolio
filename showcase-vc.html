VirtuaCade is a QML/QT based GUI developed with the intent of creating a console-like experience for my custom arcade cabinet build, mini-pc and other future gaming devices. VCGUI is colourful, highly optimised and designed for easy readability on all types of screens.

PegasusFE serves as the backend for reading and formating game metadata which VCGUI then filters, sorts and displays. VCGUI also extends functionality of Pegasus and QT Quick like seamless looping music, scalable highlight borders and the ability to list multiple launchable files under a single game entry.

Additional Features

Menu Music Jukebox

Supports an unlimited number of music tracks through a simple metadata file and even allows for seamless intro->loop music which the default QT implementation does not.

Dark Mode

Instantaneously toggleable night mode, can also be automatically set based on time of day.

Power Controls

VCGUI can shutdown and restart your device, eliminating the need to deal directly with the operating system.

Animated Game Backgrounds

VCGUI extends Pegasus' video function to allow for animation code to be directly executed, allowing for better performance, adaptability, image quality and file size

View Game Controls

Control information can be added into a game's metadata and viewed. Especially useful for arcade cabinets as the buttons tend to not resemble typical controller layout and button prompts that most games use.

Genre Specific Menus

Currently includes a special menu for fighting games and racing games for steering wheel controls.

Animations & Shader Effects

Fun animation tricks and GLSL shader effects make the interface come alive.
